# LLM Agent Experiments

This repository contains experiments exploring LLM behavior, prompt engineering, and agent-style interactions using models such as Claude, ChatGPT, and NotebookLM.

## Focus Areas
- Prompt structure and response reliability
- Context injection and memory strategies
- Guardrails for safer AI responses
- Experiments with conversational AI agents

## Background
These experiments were informed by my work building an AI-powered conversational system for a mental wellness platform, along with my broader interest in responsible AI, cybersecurity, and human-centered AI deployment.

## Goals
The goal of this repository is to document practical experimentation with LLM systems and to better understand how prompt design, memory, and safety constraints affect model behavior in real-world applications.
